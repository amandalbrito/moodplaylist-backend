# 🎧 MoodPlaylist

## 📌 Sobre o projeto
O MoodPlaylist é uma aplicação web que utiliza inteligência artificial para recomendar playlists com base no humor do usuário.

A proposta do projeto é unir experiência do usuário com tecnologia, oferecendo recomendações personalizadas de forma simples e intuitiva.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- TypeScript
- Python (IA)
  
## 🎯 Funcionalidades
- Identificação de humor do usuário
- Recomendação de playlists personalizadas
- Interface interativa e responsiva

## ▶️ Como executar o projeto
```bash
# Clonar o repositório
git clone https://github.com/amandalbrito/moodplaylist-backend

# Rodar backend (Python)
python app.py

# Abrir frontend no navegador
http://localhost:550
```
